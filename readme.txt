One WooCommerce installation to many external websites. Display stores on as many websites as required from single WooCommerce install

== Description ==
Install WooCommerce on one site and show the same store on as many websites as needed without having to do additional WooCommerce installation on other external sites.

There is an option to choose whether to display a single product or any other number of products on the external website.

Furthermore, select only the categories that you want to show products from on the external website.

Check Documentation [Documentation](http://momothemes.com/documentationwooframe/ "Documentation")

== Installation ==
= This plugin can be installed directly from your site. =
1. Log in and navigate to Plugins & Add New.
2. Type \"wcexternal\" into the Search input and click the \"Search\" button.
3. Locate the \"wcexternal\" in the list of search results and click \"Install Now\".
4. Click the \"Activate Plugin\" link at the bottom of the install screen.

= It can also be installed manually. =

1. Download the \"wcexternal.zip\" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the \"Plugins\" screen.
4. Locate \"wcexternal\" in the list and click the \"Activate\" link.
